# Speedbump Privacy Policy

## 1. What data we collect

Speedbump will collect the following data
| Type | Purpose | Retention |
| :---: | :---: | :---: |
| Server ID | To identify and store bot configuration for different servers | Until Speedbump is removed from the server |
| Channel IDs | To store slowmode rules per channel | While there are active rules on that channel |
| Slowmode Rules | To properly handle automatic slowmode changes | Until that rule is deleted |
| Message Timestamps | To detect channel activity | Cached in RAM, until the cache expires |

No personal data or message content is ever stored or even processed

## 2. How do we use this data

1. To adjust slowmode settings to channels in your server
2. To decide if a rule should be triggered based on activity

We do not share data with 3rd Parties

## 3. Data retention and deletion

- Rules are stored in a database until removed by a server admin
- Channel IDs are stored in a database until there are no rules that refrence it
- Server IDs are stored in a database until Speedbump leaves the server
- If Speedbump is removed from a server, all related data is immediately removed

## 4. Updates to this policy

Updates to this policy will be announced in the [Discord Support Server for Speedbump](https://discord.threadedapp.xyz)
